🌤️ Weather Data API

A simple and fast interface for accessing historical temperature data.

How to use (Endpoints)

  • Specific date: /api/v1/[station_id]/[date]
  • All data: /api/v1/[station_id]
  • Yearly data: /api/v1/yearly/[station_id]/[year]

Example: /api/v1/10/1988-10-25
